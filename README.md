# Student Performance Prediction using Machine Learning

## Project Overview

This project builds and evaluates machine learning models to predict student math scores using demographic and academic performance data.

## Objectives

- Data Preprocessing
- Feature Selection
- Train/Test Split
- Model Training
- Model Evaluation
- Performance Comparison

## Dataset

Student Performance Dataset

## Machine Learning Models

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Evaluation Metrics

- RMSE (Root Mean Squared Error)
- R² Score

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Files Included

- Student_Performance_Prediction.ipynb
- StudentsPerformance.csv
- ML_Report.pdf
- README.md

## Conclusion

The project compares multiple machine learning models and identifies the most effective model for predicting student performance.
